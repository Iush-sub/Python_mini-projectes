# To-Do List CLI App

A simple command-line To-Do List application built with Python. Tasks are stored in a JSON file, allowing data to persist between sessions.

## Features
- Add tasks
- View tasks
- Update task status
- Delete tasks
- JSON data storage

## Requirements
- Python 3.x

## Installation

Clone the repository:

git clone https://github.com/Iush-sub/todo-app.git

Move into the project directory:

cd todo-app

Run the application:

python main.py
